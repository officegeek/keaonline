# Semantic Model

## Steps for Import til Power BI Services

- **Oprette forbindelse til dine datakilder**
    - *CSV, Excel, eller...*
- **"Rens" dine data**
    - *Fjern tomme rækker*
    - *Fjern dubletter*
- **Oprete relationer mellem tabeller**
    - *Identificer Primær og Fremmede nøgler*
- **Opret beregninger**
    - *Evt. med "fletet" data*
- **Publicer til Power Bi Services**
